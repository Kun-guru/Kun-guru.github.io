    Home - > Favorites

    A favorite is a quick link to a resource that you access frequently to complete own work. It can be a link to a file or a program, a bookmark, or a saved search that is created within the IWMS Client. You can manage your own favorites. If you add a favorite, it is available only to you

    Manage Favorites

    Name

    shows the list of already created favorites, the buttons below are used to manage these favorites and create new ones

    New Link

    1. Click the manage favorites button to open the manage favorites window
    2. Type the desired name for your favorite in the link name input box
    3. For the link path, this field will be filled after selecting one option among browse to a program file, Link to bookmark, Link to saved search
    4. Browse to a program file – opens windows explorer where user can select any windows program shortcut. After selecting the desired windows application, the application link will be filled in the link path
    5. Link to bookmark -
    6. Link to saved search – [note: you must have an already saved search for this option to work, to save a search go to work management, do a search and save after getting desired results] opens a window which has saved search results, select any of the saved searches and click OK to proceed. You can also remove undesired saved searches in this window by selecting any saved search and using the delete option
    7. After successfully adding a name and selecting a link, click on the update link then OK
    8. The created favorite will be visible in the favorites section

    Delete Link

    Deletes an already created favorite link and removes it from the favorite list

    Update Link

    Saves the changes made on a link, and updates in the favorites list after OK button is pressed. Changes can be rename link name or change link path.

    Up

    [only active when there are multiple items in the list and there is an item above the selected one]Change the position of a favorite and move it upwards, one step up on every click. The button disables automatically when the item gets to the first position

    Down

    [only active when there are multiple items in the list and there is an item below the selected one] Change the position of a favorite and move it downwards, one step down on every click. The button disables automatically when the item gets to the last position down

    Home -> Dashboards

    The Dashboard section displays the statistical information related to the inbox and also lists the pending activities that you have to complete. It also displays Today's Work, Current Work, and Future Work which will show Items and the count.

    Inbox statistics

    The Inbox Statistics section displays the work items in your Inbox. It provides vital information about the work items such as the work type, the region of the work, the total items of a work item based upon area and work type, any tasks that need immediate attention (jeopardy), and tasks where the due date has already occurred. Clicking a row in the Inbox Statistics section displays the Inbox.

    Today’s work

    Displays the summary for the work assigned for today in a tabular form, with the details of the respective work items and their count. The work items are; notifications (un-assigned), Trouble Orders (Assigned), Trouble Orders(un-assigned), work orders (assigned), and work orders (un-assigned). Each of the order types is displayed with the total number of such orders available. Clicking on any of the items in the table rows i.e. the work items listed above, leads to the inbox section where there are more details on the work items.

    Current work

    Displays the summary for the work in progress in a tabular form, with the details of the respective work items and their count. The work items are; notifications (un-assigned), Trouble Orders (Assigned), Trouble Orders(un-assigned), work orders (assigned), and work orders (un-assigned). Each of the order types is displayed with the total number of such orders available. Clicking on any of the items in the table i.e. the work items listed above, leads to the inbox section where there are more details on the work items.

    Future work

    Displays the summary for assigned upcoming work items in a tabular form, with the details of the respective work items and their count. The work items are; notifications (un-assigned), Trouble Orders (Assigned), Trouble Orders(un-assigned), work orders (assigned), and work orders (un-assigned). Each of the order types is displayed with the total number of such orders available. Clicking on any of the items in the table i.e. the work items listed above, leads to the inbox section where there are more details on the work items.

    Home -> Widgets

    Note: The widgets are visible as per the role assigned to you. Widgets are assigned a role by an Administrator from the Web Portal.

    Widgets are quick tools providing shortcuts and a summary of important items to the user. The system consists of six widgets. The widgets are assigned to a role. A user who is assigned a role can view the assigned widgets.

    Messages

    The Messages widget displays the tally of unread messages and total messages in your Inbox. Clicking the Messages widget redirects to the inbox and opens the messages.

    Pending alerts

    The Pending Alerts widget displays the tally of unread alerts and total alerts in your Inbox. The tally of the pending alerts changes as per the settings made on the preferences under the general tab. Clicking the pending alerts widget displays the list of pending alerts on the Inbox page. An alert is a system-generated message such as a released timesheet alert

    Last sync

    The Last Sync widget displays the date and time when your local database was last synced with the server database. When the Sync operation happens, any data that is locally saved by you is uploaded to the server, and simultaneously, any new data on the server is downloaded to your local database. You can configure sync at fixed intervals or can sync manually.

    Search

    The Quick Search widget enables you to search objects in the system. The Quick Search widget displays a list of object types. The list in the quick search widget is displayed as per the user role.

    To use the quick search widget: [what are new map search results, need some searchable items to be able to see the results]

      1. select the category for the desired search record
    • Type the first four characters of the record
    • Select the desired subcategory, the search will then begin
    • The system displays the asset on the map, or if the asset is not found on the map, an error message window is displayed.

    Date/Time

    The date/time widget displays the current date and time

    Available AORs

    AOR implies Area of Responsibility. The Available AORs widget displays the list of areas. You can search, view, subscribe, or be assigned tasks from the areas in the list.

    Home -> Pending Activities

    The pending activities widget contains a summary of activities that are pending. When you click on a row in the list, you will be redirected to the screen with a full list of the items in the pending list. For example, if we have notifications in the pending activities list, then on clicking on the notifications row, the user will be redirected to the notifications screen.

    Inbox

    Inbox -> Assigned Work

    The Assigned Work folder displays messages of work items that are assigned to you by the Supervisor. The folder displays work according to the work type and the completion status. For every work type, there is a set of two folders. For example, My Work Orders and My Completed Work Orders, My Trouble Orders and My Completed Trouble Orders. The folders are displayed in the Inbox based on the type of work assigned to you.

    The first-level folders under a category are the system-created folders. You can also create additional folders by right-clicking a folder. The additional folders will be displayed right under the default folders depending on the folder which the user wants to create a sub-folder for. You can drag or right-click on items to add them to the additional folders.
